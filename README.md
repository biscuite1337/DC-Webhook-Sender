Here's a quick tutorial on how to use it (even if it's pretty straightforward).

**1. Webhook URLs**
This is where you'll paste the webhooks you want to send something to. Keep in mind that you need to separate them with a comma `,` otherwise it will not work.

**2. Identity**
This is where you'll input your bot name (it can be anything). You can also add an image, however keep in mind that the image must be under 10MB, otherwise you might get an error. Even if it's under 10MB, sometimes it might still give an error — in that case, just proceed without setting an image for the bot. I'm sorry for this flaw, however it's due to Discord API and payload formatting limitations.

**3. Message**
This is where you'll type your messages or links that you want to send to the mentioned webhooks. If you want to mention specific users in the server, use `<@their\_discord\_id>`.

**4. Cooldown**
This controls how fast the messages are sent to the webhooks. Keep in mind that sometimes it might get rate limited by Discord. I personally recommend using a delay of at least **0.60 seconds**, however you can play around with it and see if it works even lower than that.

**5. Start/Stop**
Self explanatory ;)

Have fun with it! ;)
